# 差旅报销单接口文档

## 1、报销单分页查询

### 接口说明

用于 5.1.1 报销单列表页面分页查询。查询条件可以全部为空，也可以按条件组合查询。

其中：

- 报销单号、标题、事由是模糊查询。
- 费用归属公司、报销部门、报销人、业务类型是精确查询。
- 当前报销单主表 `fk_reim_main.reim_company_names` 保存费用归属公司名称汇总，用于列表页展示和查询。权威分摊数据仍然是 `fk_reim_allocation`，创建/更新分摊时需要同步回写主表缓存字段。

### 请求地址

```http
GET /api/reimbursements/page
```

### 请求参数

| 参数名 | 类型 | 是否必填 | 查询方式 | 说明 |
| --- | --- | --- | --- | --- |
| pageNum | Integer | N |  | 当前页，默认 1 |
| pageSize | Integer | N |  | 每页条数，默认 10 |
| reimNo | String | N | 模糊 | 报销单号，例如输入 `100` 可匹配包含 `100` 的单号 |
| title | String | N | 模糊 | 报销标题 |
| reason | String | N | 模糊 | 报销事由 |
| reimCompanyName | String | N | 精确 | 费用归属公司名称；查询主表缓存字段 `reim_company_names` |
| reimDepartmentName | String | N | 精确 | 报销部门名称；对应主表 `reim_department_name` |
| reimburserKeyword | String | N | 精确 | 报销人姓名或工号；匹配主表 `reimburser_name` 或 `reimburser_no` |
| businessTypeName | String | N | 精确 | 业务类型名称；对应主表 `business_type_name` |

### 查询逻辑

1. 主查询表是 `fk_reim_main`。
2. `reimNo`、`title`、`reason` 使用 `LIKE` 查询。
3. `reimDepartmentName` 精确匹配 `fk_reim_main.reim_department_name`。
4. `reimburserKeyword` 精确匹配 `fk_reim_main.reimburser_name` 或 `fk_reim_main.reimburser_no`。
5. `businessTypeName` 精确匹配 `fk_reim_main.business_type_name`。
6. `reimCompanyName` 查询主表缓存字段 `fk_reim_main.reim_company_names`。因为一个报销单可能有多个费用归属公司，实际实现时建议用包含匹配或按约定拆分匹配。
7. 默认按 `created_at DESC` 排序。

### 响应字段

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| total | Long | 总记录数 |
| pageNum | Integer | 当前页 |
| pageSize | Integer | 每页条数 |
| records | Array | 当前页数据 |

`records` 字段：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| id | Long | 报销单主表 ID |
| reimNo | String | 报销单号 |
| billStatus | String | 单据状态编码 |
| billStatusName | String | 单据状态名称 |
| billTypeName | String | 单据类型名称 |
| reimburserName | String | 报销人姓名 |
| reimburserNo | String | 报销人工号 |
| reimDepartmentName | String | 报销部门名称 |
| reimCompanyNames | String | 费用归属公司名称；多个公司可用逗号拼接 |
| businessTypeName | String | 业务类型名称 |
| title | String | 报销标题 |
| reason | String | 报销事由 |
| allowanceAmount | BigDecimal | 补助金额 |
| createdAt | LocalDateTime | 创建时间 |
